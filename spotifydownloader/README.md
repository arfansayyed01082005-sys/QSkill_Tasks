Task Name 2:

Spotify Song Downloader Using RapidAPI

Description:

This task is a web-based Spotify song downloader developed using HTML, CSS, and JavaScript.

Users can paste a Spotify song link, and the application uses RapidAPI to generate a downloadable link for the song.

Functionality:

* Accepts Spotify song URL from user

* Sends request to RapidAPI

* Fetches downloadable song link

* Displays download option

* Shows alerts for success and error

Concepts Used:

* HTML Input Forms

* CSS User Interface Design

* JavaScript Fetch API

* REST API Integration

* JSON Handling

* Asynchronous Programming

How to Run:

1. Open the project in VS Code

2. Add your RapidAPI key in script.js

3. Open index.html using Live Server

4. Paste Spotify song link and click Download

Example Input:

https://open.spotify.com/track/4uLU6hMCjMI75M1A2tKUQC

Example Output:

Alert: Download link generated successfully

Download Link displayed on screen
